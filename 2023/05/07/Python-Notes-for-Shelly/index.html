背景概述

近段时间，“孔乙己的长衫”（用来指代找不到体面工作的人的所谓“高学历”）成为网络平台热门话题。本人希望借助python程序设计技术，了解网络平台（B站）的话题舆论趋势和导向。

（本质上，是python课程设计作业，还有各种要求。正是因为各种要求，代码里面有部分所谓“脱裤子放屁”的地方。）

具体流程

  1. 在哔哩哔哩平台找到5个有关“孔乙己的长衫”关键词的视频，记录下视频编号，即bv号，分别是BV1s54y1M7wQ, BV18s4y1n77h, BV1xT411k7Ce, BV1Eb411Q7rs, BV1qT411q7D2，以字符串的形式保存在一个列表中。
  2. 将视频评论区内容通过手段获取后保存至数据库。
  3. 对数据库内容进行分析，包含统计词频；调用网络上下载的模块制作词云、分析情感等。
  4. 输出结果

10.面向对象

在这个部分，我们首先需要了解面向对象的思想方法，这个思想方法主要包含两个方面。

  1. 一个对象，包含属性和行为，分别对应一个类里面的变量和方法。
  2. 对象是一个具体的、实例化的概念。与之相对的概念是类。两者是具体和一般的关系。实例化后的对象用一个变量进行存储。
